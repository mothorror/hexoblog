## 准备工作

### 硬件要求

- **CPU**：Intel 8 代及以上（推荐）/ AMD Ryzen 及以上
- **显卡**：AMD 显卡优先 / Intel 核显
- **内存**：8GB 及以上，推荐 16GB
- **无线网卡**：博通系列（免驱）/ 英特尔（需要补丁）
- **蓝牙**：博通或英特尔（建议购买免驱版本）

### 软件准备

- **macOS 镜像**：从 App Store 下载或获取 DMG 文件
- **OpenCore 引导**：从 [OpenCore 官方](https://github.com/acidanthera/OpenCorePkg) 下载最新版本
- **BalenaEtcher** 或 **Rufus**：制作启动盘
- **Hackintool**：硬件检测和配置工具
- **ProperTree**：编辑 config.plist

## 安装步骤

### 1. 制作启动盘

使用 BalenaEtcher 将 macOS 镜像写入 U 盘（建议 16GB 以上）。

### 2. 配置 EFI

1. 解压 OpenCore 包到 EFI 分区
2. 使用 ProperTree 编辑 config.plist
3. 根据硬件配置 ACPI、DeviceProperties、Kernel 等

### 3. BIOS 设置

| 设置项 | 推荐值 |
|--------|--------|
| 启动模式 | UEFI |
| 安全启动 | 关闭 |
| VT-x/AMD-V | 开启 |
| Above 4G Decoding | 开启 |
| DVMT Pre-Allocated | 64MB |

### 4. 安装 macOS

1. 从 U 盘启动
2. 选择 "Install macOS"
3. 在磁盘工具中抹盘（建议 APFS 格式）
4. 完成安装

### 5. 安装后配置

1. 再次从 U 盘启动，进入已安装的系统
2. 安装必要的 Kexts
3. 修复声卡、网卡等驱动
4. 配置双系统引导（如需要）

## 常见问题

### Q1：卡在 "++++++++++++"？

A：可能是 config.plist 配置问题，检查 ACPI 和 Kernel 设置。

### Q2：无法进入安装界面？

A：检查显卡注入设置，尝试使用 ig-platform-id。

### Q3：安装后无法启动？

A：检查 EFI 是否正确放置，确保 BOOTX64.efi 在正确位置。

## 总结

黑苹果安装需要耐心和细心，遇到问题多查阅资料和社区讨论。祝安装顺利！
